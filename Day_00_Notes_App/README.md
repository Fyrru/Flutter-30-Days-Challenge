# 🚀 Flutter 30 Days Challenge

Мой личный марафон по изучению Flutter. Каждый день — новое приложение, новый навык и чистый код. 

---

## 📋 Прогресс марафона

| День | Проект | Описание / Изученные навыки | Статус |
|:---:|:---|:---|:---:|
| 00 | [Notes App](./Day_00_Notes_App) | Заметки на Hive, работа со Stack, анимации и рефакторинг | ✅ Готово |
| 01 | **Day 01: Portfolio** | Создание визитки, работа с CircleAvatar и внешними ссылками | ⏳ Завтра |

---

## 🛠 Технологии
* **Язык:** Dart
* **Фреймворк:** Flutter
* **База данных:** Hive (локальное хранение)
* **Архитектура:** Разделение по слоям (Models/Screens)

---

## 📱 Как запустить
1. Склонируйте репозиторий: `git clone https://github.com/ТВОЙ_ЛОГИН/Flutter-30-Days-Challenge.git`
2. Перейдите в папку нужного дня.
3. Выполните `flutter pub get`.
4. Запустите проект: `flutter run`.
